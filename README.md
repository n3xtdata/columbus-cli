# columbus-cli

## Development

## Setup Go and Dep

``` shell
brew install go
```

``` shell
brew install dep
```

``` shell
export GOPATH=$HOME/go
```

## Get the columbus-cli github project

``` shell
go get -v github.com/n3xtdata/columbus-cli
```

``` shell
cd $HOME/go/src/github.com/n3xtdata/columbus-cli
```

## Build

``` shell
make build
```

## Install

``` shell
make install
```

## Use the columbus cli

``` shell
columbus help
```
